"Se o conceito de valor é considerado, então o objeto real é visto só como um signo, não vale por si mesmo, mas pelo que vale."(MARX, 2013, p.118)

O desvelamento do conceito de valor em Marx é fundamental para a compreensão do que é mercadoria, já que é no conceito de valor que está o segredo, a metafísica que encanta as mercadorias. Quando Benjamin aproxima as mercadorias das alegorias, é o valor que ocupa o campo de significação. O valor não é algo natural, não é uma medida quantitativa que encontra-se mensurável no corpo da própria mercadoria, como o peso, tamanho, ou seu valor de uso, trata-se de uma quantificação por meio de uma matéria oculta e complexa implicada nas relações sociais de trabalho. Trata-se, portanto, de leis de uma outra natureza.
